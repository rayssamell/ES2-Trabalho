**Sistema de Vendas**

Contempla cadastro de produtos, pedidos e itens.

 - O Script SQL está em schema.sql
 - SGBD: Postgres
 - Tecnologia: Java Hibernate
 - Interface simples: HTML

**Para executar**

Inicie o servidor com mvn exec:java 
Enquanto o servidor roda, entre em index.html para visualizar os produtos e pedidos do banco de dados.
Você pode cadastrar um produto na tela de produtos e um pedido na tela de pedidos.
